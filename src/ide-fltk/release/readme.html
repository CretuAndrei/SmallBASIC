Features

  • Based on FLTK library version 2.x. FLTK is a collection of inter-working programs (library) written in the C++ programming language. It allows a programmer to make their graphical user interface (GUI) programs run on various operating systems including Linux, Mac and Windows.
  • Reduced CPU utilisation while a program is running.
  • Syntax high-lighting:
    keywords functions procedures comments
  • New/updated commands:
    IMAGE Allows you to display an .GIF image with code such as: 
    open "logo.gif" for input as #1
image #1, 0, 1,25
close #1

    Syntax: IMAGE handle, index, x, y [,sx,sy [,w,h]]   (index not used in this release)
    IMAGEW Returns the width of an image
    IMAGEH Returns the height of an image
    HTML Displays html text in a "modeless" frame. The html command returns immediately while the output remains displayed. Syntax: HTML html, [title,] [x,y,w,h]   (title not used in this release). Anchor links can be used to open or edit another SmallBASIC program once the currently running program has completed.
    BUTTON Syntax: BUTTON x, y, w, h, variable, caption [,type]
    type can optionally be 'radio' | 'checkbox' | 'link' | 'label' variable is set to 1 if a button or link was pressed (which will have closed the form, or if a radio or checkbox was selected when the form was closed.
    TEXT Syntax: TEXT x, y, w, h, variable
    When DOFORM returns the variable contains the user entered value.
    DOFORM syntax: DOFORM [x,y,w,h]
    Executes the form.
  • Support for modules. The module directory is specified on the command line using the -m switch.
  • Support for Plugins. Allows the editor to be extended with programs written in SmallBASIC. An alternative plugin home-directory may be specified on the command line using the -t switch.
